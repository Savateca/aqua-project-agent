1) Copie os arquivos deste pacote para dentro da pasta raiz do projeto, mantendo a estrutura de pastas.

2) Se as pastas core_engine e data_access ainda não existirem no seu projeto, apenas copie-as do pacote.

3) No PowerShell, dentro da pasta do projeto, rode:

.\venv\Scripts\python.exe -m streamlit run app.py

4) O sistema criará automaticamente uma pasta chamada projects/ para salvar os projetos em JSON.

5) Na barra lateral do app agora existem as opções:
- Abrir projeto
- Novo projeto
- Duplicar projeto
- Excluir projeto
- Salvar projeto completo (na aba Dashboard)
