1) Copie o conteúdo do pacote por cima da pasta do projeto.
2) Renomeie infra/env.example para .env e preencha as chaves do Supabase.
3) Execute infra/supabase_schema.sql no SQL Editor do Supabase.
4) Instale dependências:

.\venv\Scripts\python.exe -m pip install -r requirements_supabase_beta.txt

5) Rode o app:

.\venv\Scripts\python.exe -m streamlit run app.py

6) Na barra lateral, escolha: Modo de armazenamento -> Supabase beta
7) Faça login com o usuário criado no Supabase Auth.
